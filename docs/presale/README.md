## 📘 Overview

The **IO Beats DAO Presale Program** distributes 2B IOB tokens (4% of total supply per chain)  
through **five rounds** combining:
- 💎 Token allocation  
- 🎨 NFT utility  
- 🗳 DAO governance rights  
- ⏳ Vesting rewards via Sablier  

Each chain’s presale aims to raise **$1 M USD** to bootstrap liquidity for DAO vaults, staking, and reward pools.

All rounds are **audited by Cyberscope**, governed via **Snapshot**, and executed through **Safe multisigs**.
